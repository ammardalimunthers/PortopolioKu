Modul: Pentingnya Portofolio Personal Web

Untuk Acara Web Basic Developer

1. Pengantar: Pentingnya Portofolio Personal

Dalam dunia pengembangan web, portofolio personal adalah alat penting untuk menunjukkan kemampuan Anda kepada calon klien atau majikan. Portofolio web memungkinkan Anda untuk:

  • Menampilkan proyek-proyek yang telah Anda buat
  • Mendemonstrasikan keterampilan teknis Anda
  • Meningkatkan kredibilitas sebagai developer
  • Membuat kesan pertama yang baik
Tips: Portofolio yang baik harus responsif, cepat dimuat, dan mudah dinavigasi.

2. Cara Membuat Website dengan HTML, CSS, dan JavaScript

Untuk membuat website sederhana, Anda perlu memahami tiga komponen dasar:

HTML (HyperText Markup Language)

HTML digunakan untuk struktur konten. Contoh:



Judul Halaman

Selamat Datang


Ini adalah paragraf.

CSS (Cascading Style Sheets)

CSS digunakan untuk styling. Contoh:

body { font-family: Arial; }
h1 { color: blue; }

JavaScript

JavaScript digunakan untuk interaktivitas. Contoh:

function greet() { alert('Halo!'); }

Langkah-langkah membuat website:

  1. Buat file HTML dengan struktur dasar
  2. Tambahkan CSS untuk styling
  3. Tambahkan JavaScript untuk fungsionalitas
  4. Test di browser

Langkah Lengkap Membuat Website Portfolio seperti yang Kita Buat

Berikut adalah langkah-langkah detail untuk membuat website portfolio seperti yang telah kita buat sebelumnya:

  1. Persiapan: Buat folder baru untuk proyek, misalnya "portfolio".
  2. Buat file HTML (index.html): Struktur dasar HTML dengan header, main, dan footer. Sertakan link ke CSS dan JS.
  3. Buat file CSS (style.css): Tambahkan styling untuk layout, warna, font, dan responsivitas.
  4. Buat file JavaScript (script.js): Tambahkan fungsi untuk interaktivitas seperti edit profil, tambah proyek, dll.
  5. Test dan Debug: Buka index.html di browser dan periksa fungsionalitas.
  6. Deploy ke GitHub: Buat repository baru dan upload file-file tersebut.

3. Cara Posting Website ke GitHub

GitHub Pages memungkinkan Anda untuk hosting website gratis. Langkah-langkah:

  1. Buat akun GitHub jika belum ada
  2. Buat repository baru dengan nama username.github.io
  3. Upload file HTML, CSS, dan JS ke repository
  4. Aktifkan GitHub Pages di settings repository
  5. Akses website di https://username.github.io
Catatan: Pastikan file utama bernama index.html.

Alternatif: Gunakan Git untuk push kode dari lokal ke GitHub.
